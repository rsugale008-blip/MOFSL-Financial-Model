# Motilal Oswal Financial Model
### Automated Investment Banking Analysis | Python + Excel

## Overview
Full institutional-grade financial model built in Python
with automated data fetching via yfinance.

## Model Components
| Sheet | Description |
|-------|-------------|
| Assumptions | Key drivers & inputs |
| 3-Statement | Income, Balance Sheet, Cash Flow |
| DCF | Discounted Cash Flow valuation |
| LBO | Leveraged Buyout — 2.4x MOIC, 18.9% IRR |
| Comps | Trading comparables — 5 peer companies |
| Football Field | Valuation summary — BUY rating |

## Tech Stack
- Python 3.12
- openpyxl (Excel generation)
- yfinance (Live NSE data)

## How to Run
pip install yfinance openpyxl
python main.py
